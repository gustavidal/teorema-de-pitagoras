## Como o programa funciona

O programa aplica o Teorema de Pitágoras, que afirma que:

> a² = b² + c²

onde:
- a é a hipotenusa,
- b e c são os catetos.

Ao executar o programa, o usuário escolhe qual valor deseja descobrir:
- `[1] Cateto`
- `[2] Hipotenusa`

Em seguida, o programa solicita os valores conhecidos e realiza o cálculo automaticamente utilizando o método `Math.sqrt()` para obter a raiz quadrada.

No final, ele exibe todos os valores (hipotenusa e catetos) de forma organizada e clara.
